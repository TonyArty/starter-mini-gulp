#### Документация в разработке
##### Установка
`git clone https://github.com/dzmitROCK/start.git`  
`yarn install`
##### Использование
`yarn start` запуск разработки  
`yarn run production` конечная компиляция всего проекта 
##### Использованы препроцессоры
* scss и sass
* pug
